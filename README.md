# Clinic Real-Time System

## Overview
This project demonstrates real-time synchronization between a patient form and a staff dashboard using Socket.IO.

## Tech Stack
- Next.js
- React
- Socket.IO
- Tailwind CSS

## How to Run

1. Install dependencies
   npm install

2. Start socket server
   node server.js

3. Start Next.js
   npm run dev

4. Open:
   - Patient: https://patient-realtime.vercel.app/
   - Staff: https://patient-realtime.vercel.app/staff
